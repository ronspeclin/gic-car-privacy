Privacy Policy

Last updated: December 26, 2023

Important Notice:

GIC Car is an independent application designed for vehicles modified by GIC. This is not an official manufacturer application. We provide services specifically for vehicles equipped with GIC modification systems.

Service Limitations:

This application will only work with vehicles that have been properly modified and equipped with GIC. It is not compatible with standard unmodified vehicles.

Permissions Usage:

  • INTERNET: Required for connecting to car management servers and receiving real-time updates
  • ACCESS_FINE_LOCATION: Required for precise car location tracking and navigation features
  • ACCESS_COARSE_LOCATION: Required for approximate location services and region-based features
  • READ/WRITE_EXTERNAL_STORAGE: Required for storing car data, user preferences, and cached map data
  • ACCESS_NETWORK_STATE: Required to monitor network connectivity for real-time features
  • WAKE_LOCK: Required to keep the device awake during critical car operations

Data Collection and Usage:

We collect and process the following data to provide car management services:

  • Location data for vehicle tracking and navigation
  • Vehicle status and diagnostic information
  • User preferences and settings

Data Security:

We implement industry-standard security measures to protect your data. All transmitted data is encrypted using SSL/TLS protocols.

Data Collection and Processing:

  • Location Data: We collect vehicle location data only when the app is in use
  • Vehicle Data: We process vehicle status information provided by GIC
  • User Account: Basic account information for authentication

Data Storage and Protection:

  • All data is encrypted in transit and at rest
  • Vehicle location history is stored for 30 days
  • User can delete their account and all associated data

Third-Party Services:

We do not share your data with any third parties except as required for basic app functionality (e.g., maps services).

Your Rights:

  • Access your collected data
  • Request data deletion
  • Opt out of non-essential data collection
